# Utilisation
- Cloner le repo / mettre en place un workspace vide avec uniquement le fichier python à mettre en package.
- Dans ce workspace, coller sur le même chemin le [le fichier contenu dans ce repo](make_project.py).
- Dans le dossier courant, executer le code ci dessous (commande potentiellement invalide selon les versions de python, le langage terminal etc.)

```cmd
python make_project.py morpion.py
```
